Welcome to the Backoffice for GUI API Demo

🦎 GAD is a convenient tool for experimenting with tests!

Here you can practice GUI tests and API tests in a safe environment.
The page has been prepared with the purpose of learning testing.
It contains simple logic, resources, intentionally prepared errors, and various challenges.
With this configuration, we are able to simulate more challenging project realities 😉

PS. Additionally, the application has several serious design flaws - curious if you will notice them? 😉

You found a 🐛BUG in 🦎GAD?  See steps!
  1. Check version of your 🦎GAD
  2. Check if there is newer version at version check page
  3. Check the Release Notes for any mention of the bug
  4. Check the official repository issues
  5. Update your 🦎GAD if there is a newer version
  6. If the error/problem still occurs, report it to us!
    1. Got to issues
    2. Create new issue
    3. Provide as much information as possible:
      • describe the bug
      • version of your 🦎GAD
      • detailed steps to reproduce the bug
      • other details, screens etc.
    This will help us diagnose and fix the error quickly😉
